# Grafana PostgreSQL Observability Lab

Projeto de laboratório para demonstrar conhecimentos práticos em Grafana, PostgreSQL, Docker e observabilidade.

A aplicação simula eventos de uma API em tempo real usando Python, grava os dados em PostgreSQL e disponibiliza dashboards no Grafana para análise de latência, erros, volume de eventos e métricas de negócio.

## Objetivos

- Subir ambiente local com Docker Compose
- Provisionar datasource PostgreSQL no Grafana
- Versionar dashboards como JSON
- Criar painéis de observabilidade técnica e de produto
- Simular métricas de API em tempo real
- Configurar alertas básicos para erro e latência
